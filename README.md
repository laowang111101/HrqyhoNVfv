# 停车场管理系统简介 java1110

## 项目概述

停车场管理系统是一款基于B/S架构的Web应用，采用MVC模式设计，实现前后端分离。系统致力于提供便捷的停车服务与管理功能。

## 技术栈

- **后端**：Java、SpringBoot、Mybatis-Plus
- **前端**：Vue
- **数据库**：Mysql
- **开发工具**：IDEA、JDK1.8、Maven

## 系统角色

- **普通用户**：拥有个人信息管理、预定停车位、缴费信息管理等权限。
- **管理员用户**：负责用户信息管理、车辆信息管理、停泊车辆管理、车辆进出管理等功能。

## 运行环境

- **系统环境**：Windows
- **开发环境**：IDEA、JDK1.8、Maven、Mysql8.0
